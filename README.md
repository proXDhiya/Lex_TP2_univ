## compiler? (only-linux)

1. Download **Flex** : `sudo apt-get install flex -y`
2. Download **GCC** (c compiler) : `sudo apt-get install gcc -y`
3. Run **compiler.sh** : `./compile.sh "YOUR_FILE.l" "YOUR_OUTPUT_FILE.out"`
4. Press `CTRL + C` to **Exit**
